== Creative coding and badge making with sketchPatch == *****CANCELLED*****

sketchPatch is an online programming playground that uses the programming language Processing. Collaborate on animations and interactive generative art created by you from the code up! This is a fun way to learn and share computer code, even if you have never done it before. At the end of the workshop we will print of our masterpieces and make badges!

2hr workshop lead by Sophie

Kit: You will need to bring a laptop.


  • hannah @handee

== Arduino Intro == *****CANCELLED*****

A real quick intro to the Arduino Uno/LilyPad. Learn how to make a light blink, make 2 lights blink, and use a simple sensor. This Arduino introduction is transferable to the LilyPad Arduino for wearable tech, so if you have one bring it with you. MzTEK can lend Arduino Uno boards and USB cables.

2hr workshop by Sophie

Kit You will need:

  • a laptop with Arduino installed.
  • If you want to use a LilyPad Arduino bring that (with FTDI board and mini USB).


  • add your name here if you want to attend this workshop. Julie johnson
